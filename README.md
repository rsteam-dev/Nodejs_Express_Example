# Nodejs_Express_Example
백석대학교 학생을 위한 Nodejs + Express를 이용한 웹서버 예제입니다.


# 예제 설치
Windows에서 VS Code를 사용할 경우
Ctrl + `을 눌러 하단에 터미널 창 생성 후
./install.sh (Enter)

리눅스에서 터미널로 설치할 경우 
sudo sh ./install.sh  (Enter)

# 예제 실행
Windows에서 VS Code를 사용할 경우
Ctrl + `을 눌러 하단에 터미널 창 생성 후
./start.sh (Enter)

리눅스에서 터미널로 실행할 경우 
sudo sh ./start.sh  (Enter)
